# Configuration

All settings have defaults вЂ” no `.env` is required to run a basic local setup. Create `.env` to override any value.

```bash
cp .env.example .env
nano .env
```

## .env reference

| Variable | Default | Description |
|----------|---------|-------------|
| `WALLET_PASS` | *(prompted)* | LND wallet password вЂ” min 8 chars. Set to skip interactive prompt |
| `BITCOIND_RPC_USER` | `regtest` | bitcoind RPC username |
| `BITCOIND_RPC_PASS` | `regtest` | bitcoind RPC password |
| `BITCOIND_RPC_PORT` | `18443` | bitcoind RPC port |
| `BITCOIN_VERSION` | `28.0` | Bitcoin Core version to download and run |
| `MINER_WALLET` | `miner` | Wallet name used for mining and funding |
| `LND_IMAGE` | `lightninglabs/lnd:v0.20.1-beta` | LND Docker image |
| `FUND_LND1_BTC` | `8` | On-chain BTC to send to lnd1 |
| `FUND_LND2_BTC` | `3` | On-chain BTC to send to lnd2 |
| `FUND_LND3_BTC` | `6` | On-chain BTC to send to lnd3 |
| `CHANNEL_SATS` | `500000000` | lnd1в†”lnd2 channel capacity (sats) |
| `REBALANCE_SATS` | `250000000` | Amount pushed to balance lnd1в†”lnd2 |
| `LND3_CHANNEL_SATS` | `250000000` | lnd3 channel capacity to lnd1 and lnd2 |
| `RTL_IMAGE` | `shahanafarooqui/rtl:v0.15.8` | RTL Docker image |
| `RTL_PORT` | `3000` | RTL web UI port |
| `RTL_PASSWORD` | *(WALLET_PASS)* | RTL login password вЂ” defaults to `WALLET_PASS` if not set |
| `RTL_DOMAIN` | *(disabled)* | Domain for HTTPS access to RTL via nginx proxy |
| `MOSTRO_REF` | `main` | Mostro git branch or tag to build |
| `MOSTRO_NSEC_PRIVKEY` | *(prompted)* | Nostr private key (`nsec1...`) вЂ” skips prompt/generation if set |
| `MOSTRO_NPUB` | *(auto)* | Nostr public key вЂ” only needed alongside `MOSTRO_NSEC_PRIVKEY` |
| `MOSTRO_RELAYS` | `wss://nos.lol,wss://relay.mostro.network` | Comma-separated Nostr relay URLs |
| `MOSTRIX_REF` | `main` | MostriX git branch or tag to build |
| `LNURL_DOMAIN` | *(disabled)* | Domain for Lightning Address via satdress |
| `LNURL_USERNAMES` | `admin` | Comma-separated Lightning Address usernames (`admin,user2`) |
| `SATDRESS_PORT` | `17422` | satdress HTTP port |

## Ports

All services bind to `127.0.0.1` вЂ” not reachable from outside.

| Service | Port |
|---------|------|
| bitcoind RPC | 18443 |
| bitcoind P2P | 18444 |
| ZMQ blocks | 28332 |
| ZMQ transactions | 28333 |
| lnd1 P2P | 9735 |
| lnd2 P2P | 9736 |
| lnd3 P2P | 9737 |
| lnd1 gRPC | 10009 |
| lnd2 gRPC | 10010 |
| lnd3 gRPC | 10011 |
| lnd1 REST | 8080 |
| lnd2 REST | 8081 |
| lnd3 REST | 8082 |
| RTL web UI | 3000 |
| satdress | 17422 |

Public ports (only if nginx proxy is configured in `~/Server/`):

| Port | Service |
|------|---------|
| 80 | HTTP в†’ HTTPS redirect + certbot challenge |
| 443 | HTTPS в†’ RTL or satdress |
